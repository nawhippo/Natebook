Stop and Remove Local Containers (if needed):


docker stop <container_id_or_name>
docker rm <container_id_or_name>

Build Docker Images:

docker build -t iad.ocir.io/axeqcztzlnaa/websiteproject/frontend:latest -f Dockerfile .
docker build -t iad.ocir.io/axeqcztzlnaa/websiteproject/backend:latest -f Dockerfile .


Tag local Docker Images:

docker tag frontend:latest iad.ocir.io/axeqcztzlnaa/websiteproject/frontend:latest
docker tag backend:latest iad.ocir.io/axeqcztzlnaa/websiteproject/backend:latest

Check rollouts status
kubectl rollout status deployment/backend-deployment

gcloud container clusters get-credentials natebook-kubernetes-cluster --zone us-central1 --project natebook-40281

kubectl apply -f combined-deployment.yaml

kubectl get deployments
